# Install

## install Android Debug Bridge

**Debian/Ubuntu**

```
sudo apt install adb
```

**RHEL/CentOS/Fedora**

```
sudo dnf install adb
```

**OpenSUSE and SUSE Linux Enterprise**

```
sudo zypper install android-tools
```

**Arch**

```
sudo pacman -S android-tools
```

*Verifying ADB Installation*

```
adb version
```
